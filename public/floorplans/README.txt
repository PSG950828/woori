우리안과 평면도 PNG 저장 위치
===========================

이 폴더에는 1~7층 평면도 PNG 가 들어갑니다.

파일명 규칙:
  floor-1.png ~ floor-7.png

생성 방법:
  python scripts\convert_floorplans.py

원본 PDF 위치 (네트워크 공유):
  \\172.11.1.5\03 업무용 검사팀\4.기타\박승근\박승근\평면도\평면도

PNG 가 없거나 로드 실패 시, 웹앱은 자동으로 placeholder 박스를
표시하므로 변환을 미뤄도 앱은 정상 동작합니다.
